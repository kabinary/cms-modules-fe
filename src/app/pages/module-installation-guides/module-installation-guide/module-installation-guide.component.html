Installation guide

Install your {{ moduleName }} module

A step-by-step guide with screenshots from the Joomla admin panel.

  1. 1

    If you haven't bought a license yet for the {{ moduleName }} module, make sure to do it on the main page. You'll need the license key to activate the module.

  2. 2

    If you haven't downloaded the {{ moduleName }} module from Joomla Extensions Directory yet, get it from this repository (always take the latest version).

    Once the module is downloaded, go to your Joomla admin panel, navigate to System > Install > Extensions.

    Drag and drop or browse to the zip module you downloaded to install it.

  3. 3

    Navigate to Content > Site Modules, search the module, and click on it.

    On the Module tab, set Position and Status to your preference.

    On the Menu Assignment tab, select the menu items where you want the module to appear.

    On the Subscription tab, fill in the required fields.

    Heads up. Click Save & Close to persist all changes — the module won't appear on the site until you do.
