1. 1

    Make sure to enable the native Joomla User - Profile plugin. This is required for the module to work properly.

    Navigate to System > Plugins.

    Search for the User - Profile plugin and click on it.

    Set the Status to Enabled and activate the fields you want to use.

  2. 2

    Navigate to Content > Site Modules, search the module, and click on it.

  3. 3

    On the Fields Configuration tab, you can add, edit, remove, or rearrange the fields as you like. Only the fields defined in Joomla's native User - Profile plugin are supported.

    Note. Only fields declared in the native Joomla User - Profile plugin are supported. To use other custom fields, store their values in the #__user_profiles table of your database.

    For more icons, visit Font Awesome.

  4. 4

    Click Save & Close to save your changes.

You should now see the module displayed on the configured pages.
